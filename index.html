Family mruby Web Installer

ブラウザから ESP32 にファームウェアを書き込みます
Web Serial API が利用できません。Chrome / Edge / Opera で開いてください（Firefox / Safari は未対応）。

はじめる前に

  • 対応ブラウザ: Chrome / Edge / Opera (デスクトップ)
  • ESP32 デバイスを USB で接続してください
  • 本体 (fmruby-core) と子マイコン (fmruby-graphics-audio) は別々のチップです。それぞれ対応するボタンから書き込んでください
  • 初回書き込みではフラッシュ全消去が走ります

fmruby-core ESP32-S3

本体マイコン。ユーザのアプリケーションが動作する側です。16MB Flash。

fmruby-graphics-audio ESP32 (WROVER)

子マイコン。NTSC 映像出力と I2S 音声出力を担当します。本体とは UART で接続。8MB Flash。
